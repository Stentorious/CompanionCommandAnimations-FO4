# Companion Command Animations
<p align="left"> <img height="350px" src="https://staticdelivery.nexusmods.com/mods/1151/images/98662/98662-1764282612-726461228.png"> </p>

## Overview
Adds unique first-person animations when the players gives commands to companions.

Includes different animations for the following commands:
- Follow
- Hold
- Go
- Attack
- Interact

## Configuration
Tested and compatible with:
- [Unlimited Companion Framework](https://www.nexusmods.com/fallout4/mods/11829)
- [Simple Everyone's Best Friend](https://www.nexusmods.com/fallout4/mods/92314)
- [Companion Command and Tactics](https://www.nexusmods.com/fallout4/mods/11784)
- [Amazing Follower Tweaks Plus](https://www.nexusmods.com/fallout4/mods/59699)

## Requirements
- None

## Installation
Installs like any other mod, manually or using a mod manager.
